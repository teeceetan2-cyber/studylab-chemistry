# 🧪 StudyLab — Chemistry

Interactive chemistry tools built with Streamlit.

**Topics:**
- pH Calculator — convert between [H⁺], [OH⁻], pH, pOH with pH scale
- Molar Mass Calculator — compute molar mass from chemical formulas + mass/moles converter
- Stoichiometry — balance reaction proportions from given coefficients
- Ideal Gas Law — PV = nRT with interactive P-V graph
- Periodic Trend Explorer — compare atomic radius, ionization energy, electronegativity, electron affinity

### Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

Or deploy on **Streamlit Cloud**: connect this repo and set entry point to `app.py`.
